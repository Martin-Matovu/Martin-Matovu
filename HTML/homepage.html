Web Developer

I am a web developer with a passion for developing cool web applications using some amazing tools and technologies.

Technicalities

I use technologies of the JAMstack to develop fast and cool web applications. In detail, I use GitHub or Gatsby Cloud to host the codebase, Gatsby to make good use of React components, and Netlify to serve the website. I use Firebase's database API, Auth0's authentication API, and Stripe for payment management.
